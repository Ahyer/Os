# Os

### 项目简介

本项目模拟了操作系统的置换算法，支持FIFO，LRU，OPT三种置换算法。可以在实现的过程中暂停，也可以保存结果为图片，支持图标动态显示置换过程。
 
此为大学的操作系统的课程设计。
